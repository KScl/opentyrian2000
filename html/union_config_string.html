Detailed Description

A short-string-optimizing string type.

This struct allows for storing up to 15 characters (plus a terminating '\0') inline. For longer strings memory will be allocated.

The tag for this union is the last character of short_buf:

  • if '\0' then short_buf is valid,
  • otherwise long_buf is valid.
