const char * text,

FontAlignment alignment,

Draws text in a color specified by hue and value and with a drop shadow.

A '~' in the text is not drawn but instead toggles highlighting which increases value by 4.

  • like JE_dString() if (black == false && shadow_dist == 2 && hue == 15)
  • like JE_textShade() with PART_SHADE if (black == true && shadow_dist == 1)
  • like JE_outTextAndDarken() if (black == false && shadow_dist == 1)
  • like JE_outTextAdjust() with shadow if (black == false && shadow_dist == 2)
Parameters
surfacedestination surface
xinitial x-position in pixels; which direction(s) the text is drawn from this position depends on the alignment
yinitial upper y-position in pixels
texttext to be drawn
fontstyle/size of text
alignmentleft_aligned, centered, or right_aligned
huehue component of text color
valuevalue component of text color
blackif true the shadow is drawn as solid black, if false the shadow is drawn by darkening the pixels of the destination surface
shadow_distdistance in pixels that the shadow will be drawn away from the text. (This is added to both the x and y positions, so a value of 1 causes the shadow to be drawn 1 pixel right and 1 pixel lower than the text.)
