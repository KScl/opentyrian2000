\doxysubsection{Detailed Description}
A short-\/string-\/optimizing string type. 

This struct allows for storing up to 15 characters (plus a terminating {\ttfamily \textquotesingle{}\textbackslash{}0\textquotesingle{}}) inline. For longer strings memory will be allocated.

The tag for this union is the last character of {\ttfamily short\+\_\+buf\+:} \begin{DoxyItemize}
